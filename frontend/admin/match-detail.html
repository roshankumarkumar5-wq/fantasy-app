1. Lock Selections

Selection closes automatically 1 hour before the match (no manual action needed for that). Use this button if you want to lock it early for any reason.

2. Upload Final Scoresheet (PDF)

Upload the official match scoresheet as a PDF for your records. This is stored as a reference link only — you'll still enter each player's stats below using the dropdown (their names are the "key", so there's no need to look up IDs).

3. Enter Match Stats

Add a row per player who played, selecting them by name. Points are calculated automatically from the global scoring rules.

4. Finalize Match & Compute Points

Calculates every user's team total using saved stats and special player multipliers, then marks the match completed. This is blocked if any picked player is still missing stats.
